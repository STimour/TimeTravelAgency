# Contributing

Thank you for your interest in contributing to this demo project.

Guidelines
- Keep PRs small and focused.
- Run `npm install` and `npm run dev` to test changes locally.
- Follow existing code style (Tailwind utility classes, functional components).
- Add tests for new logic (Vitest + React Testing Library are included).

Reporting issues
- Open an issue with a short description and screenshots if relevant.

Pull requests
- Describe the change and why it matters.
- Link to any relevant issue.
- Ensure linting and tests pass before requesting review.
